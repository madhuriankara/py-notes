Assignment 4 Find Minimum and Maximum Write a program that:

Takes a list of numbers as input. Finds and prints the smallest and largest numbers in the list.

def get_max_min(number):
    if number > 100:
        return f'{number} is maximum'
    else:
        return f'{number} is minimum'

get_max_min(50)

'50 is minimum'

get_max_min(102)

'102 is maximum'
