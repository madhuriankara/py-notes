Assignment 5 Checks if the number is a prime number. Prints "Prime" or "Not Prime."

def get_prime_number(number):
    if number % 2:
        return f'The {number} is prime'
    else:
        return f'The {number} is not prime'

get_prime_number(3)

'The 3 is prime'
