 Check Leap Year
Write a program that:

Takes a year as input.
Checks if it is a leap year or not.
Prints "Leap Year" or "Not a Leap Year."



```python
def get_leap_year(year):
        if year % 4 ==0:
            return f'{year} is leap year'
        else:
            return f'{year} is not a leap year'
get_leap_year(2024)
```
